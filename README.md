# 🚗 تطبيق نشمي - Nashmi

تطبيق نقل ذكي في الأردن يوفر خدمة توصيل آمنة وموثوقة مع تجربة مستخدم سلسة وسهلة.

## 📱 نظرة عامة

**نشمي** هو تطبيق معاصر لطلب الرحلات يربط الركاب بالسائقين المحترفين في الأردن. يتميز بـ:
- 🗺️ تتبع الرحلة الحي
- ⭐ نظام تقييم موثوق
- 💳 طرق دفع متعددة
- 👨‍💼 لوحة تحكم للسائقين
- 🔒 أمان وخصوصية عالية

## 🛠️ التقنيات المستخدمة

### Frontend
- **Flutter** - لتطوير التطبيق الأساسي
- **Dart** - لغة البرمجة
- **GetX/Riverpod** - لإدارة الحالة
- **Google Maps** - للخرائط والملاحة

### Backend (قريباً)
- **Node.js/Express** أو **Firebase**
- **PostgreSQL/MongoDB** - قاعدة البيانات
- **Socket.io** - للتتبع الحي

### التطبيقات والخدمات
- **Firebase Auth** - المصادقة
- **Twilio/Asanpay** - SMS و OTP
- **Stripe/Fawateer** - مع��لجة الدفع
- **Google Maps API** - الخرائط والملاحة

## 📋 المتطلبات

### للبدء في التطوير
- Flutter SDK (v3.0+)
- Dart (v3.0+)
- Android Studio / VS Code
- Device/Emulator
- Git

### للتثبيت
```bash
# استنساخ المستودع
git clone https://github.com/majdmhdawe89-cloud/nashmi-jordanian-ride.git
cd nashmi-jordanian-ride

# تثبيت المكتبات
flutter pub get

# تشغيل التطبيق
flutter run
```

## 📝 اتفاقيات الكود

### تسمية الملفات
- الملفات: `snake_case` (مثال: `user_model.dart`)
- الفئات: `PascalCase` (مثال: `UserModel`)
- المتغيرات: `camelCase` (مثال: `userName`)

### التنسيق
- استخدم `dartfmt` لتنسيق الكود
- عمق البادئة: 2 مسافات
- طول السطر الأقصى: 80 حرفاً
